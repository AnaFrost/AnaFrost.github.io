О приложении

Это приложение представляет собой компонент карточки написанный на нативном Java Script и имеет следующие элементы:

  • Меню миниатюр с изображениями продукта, по которым можно переключать основное изображение
  • Блок с информацией о товаре который имеет заголовок, описание товара, цену за единицу товара, цену без скидки, количество товара в наличии и кнопку добавления товара в корзину
  • Кнопка добавления товара активна, если товар в наличии. Так же в корзину можно добавить столько единиц товара, сколько единиц имеется в наличии
  • Кнопка корзины на которой отображается количество добавленных в нее товаров
